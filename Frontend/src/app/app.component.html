🔧 Technologies Behind the Chatbot

Explore the powerful AI stack powering real-time responses, intelligent search, and secure interactions.

🧠 Large Language Model (LLM)

Powered by OpenAI’s GPT-3.5 / GPT-4 to generate natural, human-like answers in real time.

📊 Embeddings

Text converted into vector embeddings using text-embedding-ada-002 for similarity search.

📁 Vector Database

Uses FAISS for fast retrieval of relevant chunks from PDFs or documents before answering.

📄 Text Chunking

Uses smart chunking (500 tokens + overlap) to process large documents efficiently for QA.

🔗 LangChain Integration

LangChain powers the RAG pipeline with retriever + LLM orchestration for grounded answers.

🔒 Flask Backend

Lightweight Python Flask API handles PDF loading, prompt construction, and OpenAI interaction.
